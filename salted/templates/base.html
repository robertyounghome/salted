My Password Keeper

Instead of keeping all of my passwords on paper like I did for years, I decided to put my Full Stack development skills to use and create my own personal password keeper. It is a Django/Postgres application utilizing a hashed password login as well as database encryption. Bootstrap 4 and NPM for static resources. Fun project.
